# 작업일지 - 2026-04-09

## 작업 내역

### Outgoing Webhook 연결 시도 → 실패

1. `.env`에 `TEAMS_WEBHOOK_SECRET` 추가 완료
2. 서버 재시작 + Cloudflare Tunnel(`bot.sokchokim.com`) 정상 확인
3. Teams 팀(AI개발팀)에 Outgoing Webhook "호비" 등록 완료
   - 콜백 URL: `https://bot.sokchokim.com/api/webhook`
   - Secret 발급 → `.env` 반영
4. 서버 POST 테스트 정상 (HMAC 없이 보내면 `Unauthorized` 응답 확인)

### 결과: `@호비` 자동완성 안 됨

- 채널에서 `@` 입력 시 "호비"가 자동완성 목록에 안 뜸
- Teams 재시작, 웹 버전 모두 동일
- **원인: 팀 소유자 권한 미보유** — Outgoing Webhook은 소유자만 정상 등록 가능한 것으로 판단
- 소유자에게 요청 불가 상황 (협조 안 됨)
