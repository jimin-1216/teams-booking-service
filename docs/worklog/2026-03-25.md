## 교훈

1. **Azure for Students 구독은 제약이 많다** — 리소스 제공자 미등록, 리전 정책 제한
2. **`azure/container-apps-deploy-action@v2`는 불안정** — az cli 직접 호출이 더 안정적
3. **서비스 주체 권한 범위 주의** — 리소스 그룹 범위 contributor는 구독 레벨 작업 불가
4. **Teams Dev Portal 클라이언트 비밀 신뢰 불가** — `az ad app credential reset` 사용
5. **Dockerfile `--ignore-scripts` 주의** — 네이티브 모듈(better-sqlite3) 빌드 안 됨
6. **웹 채팅 테스트 시 인증 변경 후 반드시 새로고침**
7. **GitHub Actions 빌드 시 환경변수가 Secrets로 덮어씌워짐** — Secrets도 같이 업데이트 필수
8. **로컬 E2E 통과 ≠ 서버 동작** — headless 환경, 메모리, 네트워크 차이 고려 필요
