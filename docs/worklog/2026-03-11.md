## 작업 내역

### 1. BookingExecutor 셀렉터 수정 (E2E 검증 기반)

- **날짜 설정**: 캘린더 방식 → 좌표 기반 `>` `<` 네비게이션 버튼으로 전환
  - React 캘린더(rdrDay)가 클릭 후 닫히지 않는 문제 해결
  - 폼 열기 **전에** 날짜 이동 → 폼이 해당 날짜를 자동 세팅
  - `setBookingDate()` + `setBookingTime()` 제거 → `navigateToDate()` 신규
- **시간 설정**: React controlled component로 외부 변경 불가 → 폼 기본값 사용
  - `fill()`, `type()`, `nativeInputValueSetter`, React `onChange` 호출 모두 무효
  - 현재 시간 기준 +30분 블록이 자동 세팅되므로 별도 설정 불필요
- **회의실 선택**: `:has-text()` → `div.css-4d7k9p` 드롭다운 항목 직접 매칭
  - 기존: 부모 컨테이너까지 매칭되어 오작동
  - 변경: 정확한 드롭다운 아이템 클래스로 회의실명+건물+층 매칭
- **예약 취소**: `scrollIntoView()` + `evaluate` 기반 클릭
  - 삭제 버튼이 뷰포트 밖일 때 스크롤 처리
  - React 오버레이가 Playwright click() 차단 → `evaluate` 내 직접 클릭
  - 사이드바 네비게이션도 `evaluate` 패턴 적용
